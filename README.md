## 简介

- 参考Notion的Light主题
- 所做的工作
  - 整体配色更改
  - 样式更改
    - 编辑器样式（与[Atom OneDark](https://github.com/zqhjl/Siyuan-Atom-OneDark)主题大部分一致）

      - 主要是各类页签列表的悬浮、选中、点击等效果
  - 整体字体更换
